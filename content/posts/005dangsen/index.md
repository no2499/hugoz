## 三商巧福 2025/12/09 開始的週年慶首日活動
憑身分證之中有連續數字"3375"(諧音三商巧福？) 可以獲得牛肉麵乙碗，是我看過最小氣的行銷活動了，這實在太少人符合資格，這活動的企劃一定是個激怒戰士。  
  
## 正文開始：符合資格的機率為何？
身分證的組成是：
區域碼 (英文單字)+性別 (男 1/女 2)+7 碼數字 +1 碼檢查碼  
  
**先計算台北市男性機率：(到時候擴充到全台兩性也是會同樣機率)**  
若要符合資格必須是以下的排列  
```
A1(3375)**** => 10^4 => 一萬組  
A1*(3375)*** => 10^4 => 一萬組  
A1**(3375)** => 10^4 => 一萬組  
A1***(3375)* => 10^4 => 一萬組  
A1****(3375) => 10^4 => 一萬組  
```
  
排除特定情況 => A1(3375)(3375) 這種出現兩次重複計算的情況 => 只有一個  
  
所有可能的組成 => 10,000 * 5 - 1 = 49,999 種  

再計算所有可能的機率 10^8 = 100,000,000 種  

**最後得出的機率為 => 49999 / 100000000 = 0.00049999% 約為 萬分之五**  

（註：最後一碼的檢查碼理論上不是完全隨機，但因為此碼是跟前面的數字所產生而且這個數字足夠大，故可以當成隨機數字來計算）  
  
如果全台灣現在居住人口為 2300 萬人，可能符合資格的人數約為：一萬一千五百人左右 (大約 11,499.77 人)，**如果全都抓來大約可以塞滿一個台北小巨蛋或高雄小巨蛋**。  
  
(註 2：因為身分證號碼不是完全隨機，是有一定的增加規則，所以不會完全映射到這個機率上，但不會差距太大。)
